# Develop

This page describes development for HTMLHint.

## How to build HTMLHint

1. Clone from git

        git clone git://github.com/yaniswang/HTMLHint.git

2. Install nodejs
    
        http://nodejs.org/

3. Install grunt

        npm install grunt-cli -g
        cd HTMLHint && npm install

4. Watch src and test files

        grunt watch

5. Run tests

        grunt dev

6. Build

        grunt
